# 📡 Morse Code Converter

A modern, responsive **Text ↔ Morse Code Converter** built as part of **Day 82 of the 100 Days of Code Challenge**.

Convert English text to Morse code and Morse code back to English instantly. The application includes real-time translation, Morse audio playback, an animated visualizer, downloadable output, and a complete interactive reference chart.

## 🌐 Live Demo

🚀 **Try it here:**  
https://day-82-text-to-morse-code-convertor.vercel.app/

---

## ✨ Features

| Feature | Description |
|----------|-------------|
| 🔄 **Bidirectional Translation** | Convert English text to Morse code and Morse code to English |
| ⚡ **Real-Time Conversion** | Instant translation as you type |
| 🔊 **Audio Playback** | Listen to Morse code using the Web Audio API |
| 🎨 **Visual Signal Display** | Animated dot-and-dash visualizer during playback |
| 📋 **Copy to Clipboard** | Copy translated output with one click |
| 💾 **Download Translation** | Export results as a `.txt` file |
| 🗂️ **Interactive Reference Chart** | Complete Morse code cheat sheet for letters, numbers, and punctuation |
| ♿ **Accessibility First** | ARIA support, keyboard navigation, and reduced-motion compatibility |
| 📱 **Fully Responsive** | Optimized for desktop, tablet, and mobile devices |

## 🧪 How Morse Code Works

| Symbol | Duration |
|----------|----------|
| Dot (·) | 1 unit |
| Dash (–) | 3 units |
| Gap between symbols | 1 unit |
| Gap between letters | 3 units |
| Gap between words | 7 units |

Words are represented using:

```text
/
```
